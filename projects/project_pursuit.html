MIXED REALITY / VR

Project PURSUIT

An asymmetric co-op experience combining VR and real-world projection mapping. Players work across virtual and physical realities.

01

Concept

Project Pursuit is an immersive two-player mixed-reality car chase experience. It creates a unique asymmetry between the players:

  • Player 1 (The Driver): Sits in a physical car setup wearing an HTC Vive headset, using a steering wheel to escape.
  • Player 2 (The Operator): Walks around the physical room with a tracked projector, scanning the environment from above to find the driver.
02

Core Features

Dual-Player Gameplay

The Driver is immersed in a car cockpit via VR. The Helicopter Operator walks physically, using a Vive Tracker-attached projector to "scan" the room.

Virtual Projection Mapping

We implemented real-time projection alignment with physical tracking. The system handles spatial calibration and depth-based visibility, ensuring visuals projected onto the floor match perfectly with the VR world.

03

Custom Volumetric Fog

Implemented with Unity’s URP Render Feature. The Volumetric Fog combines baked lighting and raymarching to simulate light scattering inside fog volumes efficiently for VR.

  1. Lighting data is sampled from Light Probes using spherical harmonics (SH).
  2. Data is stored in a 3D Texture (Baked Light Volume).
  3. At runtime, the shader raymarches through this volume, accumulating scattering and transmittance.
